# 📖 About the Project

This project analyzes **medical examination data** to explore relationships between cardiovascular disease, body measurements, blood markers, and lifestyle habits.

Using **Pandas**, **Matplotlib**, and **Seaborn**, the project performs data preprocessing, feature engineering, statistical analysis, and creates meaningful visualizations to identify health-related patterns.

---

# 🚀 Features

✅ Calculate BMI and determine overweight patients

✅ Normalize cholesterol and glucose values

✅ Clean inconsistent medical records

✅ Generate Categorical Plot

✅ Create Correlation Heatmap

✅ Analyze lifestyle and health indicators

✅ Perform Data Cleaning using Pandas

```bash

```



